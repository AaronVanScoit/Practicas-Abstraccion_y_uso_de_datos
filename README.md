## Tipo de dato — Grafo, Dígrafo y Árbol

Los proyectos 38, 39 y 40 implementan un tipo de dato propio que contiene:

```
estructura principal {
    nodos    → conjunto de nodos  (id, etiqueta)
    aristas  → conjunto de aristas
    rutas    → conjunto de conexiones {
        conexionNodo2Nodo {
            nodoInicial
            nodoFinal
            aristaConexion
            tiempo
            costo
        }
    }
}
```

Cada proyecto:
- Lee su información desde `datos.xml` o `datos.json`
- Aplica el **algoritmo de Dijkstra** para encontrar el camino de menor costo
- Exporta los resultados en **4 formatos**: `salida.txt`, `salida.csv`, `salida.json`, `salida.xml`

La diferencia entre los tres es:
- **Grafo** → aristas no dirigidas (se puede ir en ambos sentidos)
- **Dígrafo** → aristas dirigidas (solo van en un sentido: `nodoInicial → nodoFinal`)
- **Árbol BST** → estructura jerárquica con recorridos inorden, preorden y posorden

Los datos de ejemplo representan la ruta **Casa → Insurgentes → Vasconcelos → Churubusco → Añil → Iztacalco → UPIICSA**.

## Contenido de cada Proyecto

Cada carpeta de proyecto está estructurada bajo un esquema de **programación modular**:

* **`archivo.h`** → Definición de interfaces, estructuras de datos y prototipos de funciones.
* **`archivo.cpp`** → Implementación detallada de la lógica y funciones.
* **`main.cpp`** → Punto de entrada del programa, ejecución y pruebas.

Los proyectos 38, 39 y 40 incluyen adicionalmente:

* **`datos.xml`** → Archivo de entrada con las conexiones en formato XML.
* **`datos.json`** → Archivo de entrada con las conexiones en formato JSON.

Al ejecutarse generan automáticamente:

* `salida.txt` · `salida.csv` · `salida.json` · `salida.xml`

## Guía de Instalación y Uso

### Requisitos Previos
* **Git** → [Instalar Git](https://git-scm.com/)
* **Compilador C/C++** → GCC

### Pasos a seguir

#### 1.- Clonar el repositorio

**HTTPS:**
```bash
git clone https://github.com/AaronVanScoit/Practicas-Abstraccion_y_uso_de_datos.git
cd Practicas-Abstraccion_y_uso_de_datos
```

**SSH:**
```bash
git clone git@github.com:AaronVanScoit/Practicas-Abstraccion_y_uso_de_datos.git
cd Practicas-Abstraccion_y_uso_de_datos
```

#### 2.- Compilación

Para la mayoría de los proyectos:
```bash
g++ main.cpp archivo.cpp -o programa
```

Para los proyectos de Grafo, Dígrafo y Árbol:
```bash
g++ F_grafo.cpp main.cpp -o grafo
g++ F_digrafo.cpp main.cpp -o digrafo
g++ F_arbol.cpp main.cpp -o arbol
```

#### 3.- Ejecución

**Linux/macOS:**
```bash
./programa
```

**Windows:**
```bash
programa.exe
```
O directamente desde Dev-C++.
